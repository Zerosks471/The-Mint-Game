## Key Decisions Made

| Decision | Choice | Rationale |
|----------|--------|-----------|
| **Game Name** | The Mint | Short, memorable, money-themed |
| **Platform** | Web-first | Faster iteration, simpler payments |
| **Complexity** | Mid-complexity | Balance of depth and accessibility |
| **Social** | Leaderboards + light social | Engagement without complexity |
| **Monetization** | Cosmetics + time-savers | Ethical, proven model |
| **Prestige** | Soft prestige ("Go Public") | Player-friendly reset mechanic |
| **Offline Cap** | 8h free / 24h premium | Drives engagement + premium value |
| **Visual Style** | Playful illustrated | Approachable, genre-proven |
| **Scale** | Startup (10K-100K users) | Realistic for small team |
| **MVP Scope** | Properties + Businesses | Focused, achievable scope |

## Revenue Model Summary

### Subscriptions
- **Monthly:** $4.99/month
- **Annual:** $39.99/year (33% discount)
- **Benefits:** +10% income, 24hr offline, 500 coins/month, exclusive cosmetics

### Premium Currency (Mint Coins)
- **Packages:** $0.99 to $44.99
- **Spending:** Cosmetics, boosts, name changes
- **Free Earning:** ~200-400 coins/month through gameplay

### Target Metrics
- Free to Premium: 3-5%
- Monthly Churn: <8%
- LTV: $25-50
- ARPU: $0.10-0.50
